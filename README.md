# SCOI1

Лабораторная работа №1, на 50 баллов.
Язык программирования - Python. Проект можно открыть, например, в Pycharm.
При открытии проекта появляется окно с возможностью добавить картинку(и). При добавлении они "складируются" в правой стороне окна, где к ним можно применять различные операции, а также менять их прозрачность. Картинки наслаиваются друг-на-друга, итоговый результат можно легко сохранить в любом удобном месте.


Во второй лабораторной работе (на 50-54 баллов) я реализовал градационные преобразования
изображения через интерактивную тоновую кривую.
Функция преобразования задаётся пользователем графически с помощью
контрольных точек.
Для ускорения работы используется LUT-таблица из 256 значений,
что позволяет быстро обрабатывать изображения даже большого размера.
Также реализованы две гистограммы, отображающие распределение
яркостей до и после обработки в режиме реального времени.

Лабораторная работа №3:
Данная лабораторная работа посвящена реализации алгоритмов бинаризации — перевода изображений в двухцветный вид (0 или 255) на основе яркостного порога. В рамках работы программно реализованы глобальные методы (Гаврилова и Отсу), где порог един для всего кадра, и адаптивные локальные методы (Ниблека, Сауволы, Вульфа и Брэдли-Рота), вычисляющие уникальный порог для каждого пикселя на основе анализа окружающего окна. Для оптимизации вычислений и достижения сложности $O(1)$ при расчете локальных характеристик использованы интегральные матрицы , а взаимодействие с пользователем осуществляется через графический интерфейс на Python, позволяющий динамически настраивать параметры фильтрации.

Лабораторная работа №4 представляет собой инструмент для комплексной обработки изображений, поддерживающий линейную и медианную фильтрацию с использованием квадратных и прямоугольных матриц произвольного нечетного размера. Ключевыми особенностями реализации являются использование оптимизированного алгоритма Quickselect для быстрого поиска медианы , автоматическая генерация ядер размытия по Гауссу с произвольным значением сигмы и применение метода зеркалирования при выходе матрицы за границы кадра для предотвращения краевых артефактов. Программная логика оптимизирована за счет работы с изображением как с массивом байтов, что сводит к минимуму количество обращений к пикселям , а интерактивный интерфейс позволяет пользователю свободно изменять параметры фильтров и размеры ядер непосредственно в процессе работы.

Лабораторная работа №5 - приложение для поканальной частотной фильтрации цветных изображений на основе двумерного БПФ (алгоритм Кули-Тьюки). Программа визуализирует амплитудный спектр и маску фильтра в реальном времени, позволяя гибко настраивать параметры пяти типов фильтров (НЧФ, ВЧФ, полосового, режекторного и узкополосного) для точной локализации и эффективного подавления периодических шумов.
