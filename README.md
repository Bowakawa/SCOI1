# SCOI1

Лабораторная работа №1, на 50 баллов.
Язык программирования - Python. Проект можно открыть, например, в Pycharm.

При открытии проекта появляется окно с возможностью добавить картинку(и). При добавлении они "складируются" в правой стороне окна, где к ним можно применять различные операции, а также менять их прозрачность. Картинки наслаиваются друг-на-друга, итоговый результат можно легко сохранить в любом удобном месте.
